## 2026-06-16 - Add Tooltips to IconButtons\n**Learning:** Icon-only buttons without tooltips are an accessibility issue because screen readers and sighted users lack context on what the button does. Wrapping them in Chakra UI's `Tooltip` component is a simple, effective way to improve this.\n**Action:** Always wrap `IconButton` and `MenuButton` with a `Tooltip` when they don't have text.
